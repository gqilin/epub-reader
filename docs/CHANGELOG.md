# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-02-03

### Added
- Initial release of EPUB Reader Core library
- Framework-agnostic TypeScript implementation
- EPUB file parsing from ArrayBuffer, Uint8Array, and Blob inputs
- Metadata extraction (title, author, description, etc.)
- Table of contents parsing with nested structure support
- Chapter content retrieval by index or href
- Cover image extraction
- Resource loading (images, styles, etc.)
- Complete TypeScript type definitions
- Vue 3 example application
- Comprehensive documentation and API reference

### Features
- Support for EPUB 2.0 and 3.0 formats
- XML parsing with robust error handling
- ZIP file extraction using JSZip
- Memory-efficient file loading
- Browser-compatible implementation
- Tree-shakable exports

### Documentation
- Complete README with API documentation
- Usage examples for Vue 3 and React
- Development setup instructions
- Browser compatibility information
