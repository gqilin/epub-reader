# EPUB Reader Vue 3 Example

这是一个完整的 Vue 3 EPUB 阅读器示例，使用 Element Plus UI 组件库和 SCSS 样式。

## 功能特性

### 🎨 界面布局
- **三栏布局**: 左侧目录、中间内容、右侧工具栏
- **响应式设计**: 适配桌面和移动设备
- **主题切换**: 支持浅色、深色、护眼主题

### 📚 阅读功能
- **EPUB 加载**: 支持拖拽和点击选择 EPUB 文件
- **目录导航**: 树形结构显示章节目录
- **章节切换**: 前进/后退按钮和下拉选择

### 📝 标注功能
- **高亮标记**: 多种颜色高亮文本
- **下划线**: 支持实线、虚线、点线、波浪线等样式
- **笔记**: 为选中文本添加笔记
- **书签**: 标记阅读位置

### 🎛️ 样式控制
- **字体设置**: 字体族、字号、字重
- **颜色设置**: 文字颜色、背景颜色
- **段落设置**: 行高、段间距、首行缩进
- **对齐方式**: 左对齐、居中、右对齐、两端对齐

## 技术栈

- **Vue 3**: Composition API
- **TypeScript**: 类型安全
- **Vue Router 4**: 路由管理
- **Element Plus**: UI 组件库
- **SCSS**: CSS 预处理器
- **Vite**: 构建工具

## 项目结构

```
src/
├── components/           # 组件
│   ├── EpubLoader.vue    # EPUB 文件加载器
│   ├── EpubViewer.vue    # 阅读器主组件
│   ├── TableOfContents.vue # 目录组件
│   └── SidePanelTabs.vue # 右侧标签页
├── views/               # 页面
│   └── Reader.vue       # 阅读器页面
├── router/              # 路由配置
│   └── index.ts
├── styles/              # 样式文件
│   └── global.scss      # 全局样式
├── App.vue             # 根组件
├── main.ts             # 入口文件
└── env.d.ts            # TypeScript 声明
```

## 使用说明

1. **加载 EPUB**: 启动应用后，点击左侧区域或拖拽 EPUB 文件到加载区域
2. **目录导航**: 在左侧目录树中选择章节进行跳转
3. **切换主题**: 点击右上角设置按钮选择主题
4. **添加标注**: 选中文字后使用标注工具栏进行高亮、下划线、笔记等操作
5. **管理标注**: 在右侧标签页查看和管理所有标注
6. **调整样式**: 点击样式按钮打开样式面板调整阅读样式

## 组件说明

### Reader.vue
主阅读器页面，包含：
- 顶部导航栏（标题、主题切换、侧边栏切换）
- 左侧目录栏
- 中间阅读区域
- 右侧工具栏

### EpubViewer.vue
阅读器核心组件，提供：
- 章节导航控制
- 标注功能（高亮、下划线、笔记、书签）
- 样式控制面板
- 标注管理界面

### TableOfContents.vue
目录组件，显示：
- 树形章节结构
- 章节标题和层级关系
- 点击跳转功能

### SidePanelTabs.vue
右侧标签页组件，包含：
- 笔记列表和管理
- 书签列表和管理
- 高亮列表和管理

## 样式主题

### 浅色主题
- 背景: 白色
- 文字: 深灰色
- 适合日间阅读

### 深色主题
- 背景: 深灰色
- 文字: 浅色
- 适合夜间阅读

### 护眼主题
- 背景: 米黄色
- 文字: 深棕色
- 减少眼部疲劳

## 注意事项

1. **浏览器兼容性**: 需要现代浏览器支持（Chrome 80+、Firefox 75+、Safari 13+、Edge 80+）
2. **文件大小**: 建议使用小于 100MB 的 EPUB 文件
3. **内存使用**: 大文件可能会占用较多内存，建议关闭其他标签页
4. **文件格式**: 仅支持标准的 EPUB 格式文件

## 开发说明

如需修改或扩展功能：

1. **添加新主题**: 在 `global.scss` 中定义 CSS 变量，在 `Reader.vue` 中添加主题切换逻辑
2. **自定义样式**: 修改 `global.scss` 中的 CSS 变量或组件样式
3. **扩展标注功能**: 在 `EpubViewer.vue` 中添加新的标注类型和操作
4. **添加新页面**: 在 `views/` 目录创建新组件，在 `router/index.ts` 中配置路由
